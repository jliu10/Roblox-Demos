# Roblox-Demos
A collection of feature demos intended for my YouTube channel [@ExoviteDev](https://www.youtube.com/@ExoviteDev).

Features are separated by folders, in each of the folderclient, server, and shared folders. For example, src/server/FeatureA, src/client/FeatureA, and src/shared/FeatureA.

Generated by [Rojo](https://github.com/rojo-rbx/rojo) 7.6.1.

## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "Roblox-Demos.rbxlx"
```

Next, open `Roblox-Demos.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
